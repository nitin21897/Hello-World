##                Hello World Programs in different languages
![Hello World](https://github.com/nitin21897/Hello-World/blob/master/images.png)

### C Hello World Program

```markdown

  #include<stdio.h>
  int main()
  {
    printf("Hello-world");
    return 0;
   }
```

### C++ Hello World Program

```markdown

  #include<iostream>
  int main()
  {
    std::cout<<"Hello-world";
    return 0;
   }
```
